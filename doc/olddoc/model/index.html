ALPS Project: model library

The model library contains code for quantum lattice models.

The library is still in the design and development phase. Documentation will be provided when the interfaces have stabilized.

Documentation

  • model.h includes all header files of the model library.

Example

The easiest way to learn a new library is to study examples first. We have provided examples, showing the most common usage features.
