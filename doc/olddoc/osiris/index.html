ALPS Project: alps library

The alps library contains code for (de)serialization and message passing.

We aim for the serialization part to be soon superseded by a Boost library.

Download, Configuring, Building and Installing

are discussed here.

Documentation

  • The generic hearder
    • alps.h includes all headers listed below
  • OS-specific libraries
    • osiris/os.h low level operating system call wrappers
  • Serialization
    • osiris/dump.h contains the base classes for the alps serialization mechanism.
    • osiris/xdrdump.h contains classes for serialization (e.g. into binary files) using the XDR format
  • Message passing
    • osiris/process.h contains classes describing processes and host machines.
    • osiris/comm.h contains wrappers to low level message passing routines.
    • osiris/mpdump.h contains classes for serialization into messages.
  • Support of standard library serialization
    • osiris/std/pair.h contains serialization for std::pair.
    • osiris/std/vector.h contains serialization for std::vector.
    • osiris/std/valarray.h contains serialization for std::valarray.
    • osiris/std/map.h contains serialization for std::map.
    • osiris/std/string.h contains serialization for std::string.
