Type SignalInfo

alps::scheduler::SignalHandler::SignalInfo

Synopsis

// In header: <alps/scheduler/signal.hpp>


enum SignalInfo { NOSIGNAL = =0, USER1, USER2, STOP, TERMINATE };

Description

symbolic names for signals. SIGINT, SIGQUIT and SIGTERM are mapped to TERMINATE SIGTSTP is mapped to STOP SIGUSR1 is mapped to USER1 SIGUSR2 is mapped to USER2
